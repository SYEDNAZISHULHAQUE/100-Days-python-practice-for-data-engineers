## üéØ Objective

The objective of this repository is to help data engineers strengthen Python fundamentals by solving practical problems commonly encountered in data pipelines.  
Each topic is organized into its own folder, and each folder contains **one Python file** with multiple problems and well-documented solutions.

The problems are designed to improve:
- Data processing logic
- SQL-style thinking
- PySpark-style transformations
- ETL workflow understanding

## üìÇ Modules Included

Python Basics for Data Engineers  
Contains 15 problems covering core Python concepts frequently used in data engineering, such as loops, conditionals, dictionaries, lists, modular code, and validations.

File Handling (CSV, JSON, Parquet)  
Contains 15 problems focused on reading, writing, validating, and converting data across common file formats used in data pipelines.

Data Cleaning & Validation  
Contains 15 problems related to handling null values, removing duplicates, validating schemas, enforcing business rules, and standardizing data.

Transformations & Aggregations  
Contains 15 problems covering filtering, grouping, sorting, aggregations, derived columns, and metric calculations.

SQL-like Logic in Python  
Contains 10 problems that recreate SQL concepts such as SELECT, WHERE, GROUP BY, JOIN, HAVING, and ORDER BY using Python logic.

PySpark-style Thinking (Python-based)  
Contains 10 problems designed to build intuition for PySpark DataFrame operations using pure Python.

Testing, Logging, and Configurations  
Contains 10 problems demonstrating unit testing, logging, exception handling, assertions, and configuration management.

Mini ETL Tasks  
Contains 10 end-to-end ETL-style exercises combining extraction, transformation, validation, aggregation, and loading logic.
